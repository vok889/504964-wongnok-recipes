
# Wongrok Recipe Project

My Webapp coding for DEV Pool test Round#1
## Steps to run application

Step 1: install xampp.

Step 2: Start xampp and run localhost on browser.

Step 3: Create database as Wongnok and import db from "CreateDB.sql"

Step 4: install VScode and Flask.

Step 5: extract sourcecode in folder what you want.

Step 6: Run the server by "app.py" file and opene the URL ‘localhost:5000’.
